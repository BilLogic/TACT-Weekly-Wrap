## 5. Component Styling Rules

- **No shadows.** All components are flat.
- **Border radius:** 0px or 2px (subtle).
- **Borders:** 1px hairlines using `Grid` color at 20% opacity.
- **Bento Grid:** Cells separated by 1px gaps, Paper background.
- **Corner Markers:** L-shaped 10x10px markers in Forest for emphasis.
- **Images:** Use `mix-blend-luminosity` at 90% opacity, full color on hover.
